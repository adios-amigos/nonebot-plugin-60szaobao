# 🎉 使用方法
| 指令  | 说明 |
|:-----:|:----:|
| 60s/每日新闻 | 查看今天的60秒早报 |
| 60s/每日新闻+设置 | 以连续对话的形式设置60秒早报的推送时间 |
| 60s/每日新闻+设置 小时:分钟 | 设置60秒早报的推送时间 |
| 60s/每日新闻+状态 | 查看本群的60秒早报状态 |
| 60s/每日新闻+禁用 | 禁用本群的60秒早报推送 |
